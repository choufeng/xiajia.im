## 与 AI 系列的关联

- **AI Agent 设计**：纯函数 + Side Effect 分离 → Agent 工具的可预测性
- **Prompt 工程**：Prompt 作为函数 → 组合链式调用、版本化
- **函数式 Iron Law**：不可变状态、纯函数行为 → Agent 可复现与调试

## 与 React 系列的关联

- **Hooks 模式**：本质是函数式状态管理的实现
- **useReducer**：传递 reducer 纯函数，描述状态迁移
- **React Server Components**：无副作用、可组合的组件
