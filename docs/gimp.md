# GIMP 使用

偶尔要用到图片处理， 但有没必要蛋疼的化大价钱买个正版PS， 毕竟只是偶尔用到一些简单的处理， 使用GIMP是最合适不过的选择了。


## 快捷键

- Shift + 鼠标滚轮 ： 左右滚动画布
- Ctrl + 鼠标滚轮： 缩放画布
