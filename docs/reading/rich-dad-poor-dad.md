# 《富爸爸穷爸爸》读书笔记

> 穷人为钱工作，富人让钱为自己工作——中间隔的不是收入，是一个叫「资产」的观念。

---

## 关于本书

| | |
|---|---|
| **书名** | 《富爸爸穷爸爸》（*Rich Dad Poor Dad*） |
| **作者** | 罗伯特·清崎（Robert T. Kiyosaki） |
| **首版** | 1997 年 |
| **类型** | 财商启蒙 / 观念重塑 |

清崎是日裔美国人，生于夏威夷。这本书诞生在一场全录（Xerox）推销员的破产后反思里——他亲爹是高学历的教育官员（穷爸爸），同学的爸爸是连高中都没毕业的商人（富爸爸）。两个爸爸都聪明、都勤奋、都爱他，但给他关于钱的建议**完全相反**。清崎用一生的观察，把这种「相反」提炼成了一套观念。

这本书和舍费尔的《财务自由之路》是同时代的姊妹篇，气质很不同：

- 清崎是**观念革命**：用「资产 vs 负债」一句重锤敲醒你，偏启发、偏故事；
- 舍费尔是**操作手册**：把财务自由拆成具体台阶，给数字、给比例、给动作。

读完清崎知道「该想什么」，读完舍费尔知道「明天早上该做什么」。**两本必须合在一起读，一个管心法，一个管手法。**

## 核心概念：资产 vs 负债——全书唯一重要的那句话

清崎全书最重要的一句话，就这一句：

> **资产是把钱放进你口袋的东西，负债是把钱从你口袋掏走的东西。**

就这么简单。注意——清崎的「资产 / 负债」定义**和会计学不同**。会计上，你自住的房子算资产（你名下的财产）；但清崎的判断标准只有一个：**它每月是给你带来现金，还是每月从你这里抽走现金？**

| 项目 | 会计分类 | 清崎判定 |
|------|----------|----------|
| 自住房（贷款） | 资产 | **负债**（每月还贷、物业、维修） |
| 出租房（租金 > 月供） | 资产 | **资产**（每月净流入现金） |
| 自用车 | 资产 | **负债**（折旧、油费、保险） |
| 股票 / 基金（分红） | 资产 | **资产** |
| 奢侈品、收藏 | 资产 | **负债**（除非能变现且超过持有成本） |

这个视角的杀伤力在于：**它把中产阶级的「财富幻觉」一棍子打散了。** 你以为你买了房就是有产者，清崎说——不，你只是签了一份几十年的负债合同。

> 工程师视角：清崎的定义本质是**现金流视角**，不是所有权视角。这和系统设计的视角完全一致——一个服务名义上「上线了」不算可用，**只有它每月、每天、每分钟持续产生正向价值，才算真在线。** 一次性投入 + 持续亏损，不是资产，是技术债。

## 两个爸爸：两套思维系统的对照

清崎用「两个爸爸」作为对照，不是为了讲故事，是为了**把两套思维系统并排展示**，让你看清自己内化的是哪一套：

| 维度 | 穷爸爸（高学历打工者） | 富爸爸（创业者 / 投资者） |
|------|----------------------|--------------------------|
| **对钱的态度** | 钱是赚来花的，省着用 | 钱是种子，要先种下再生钱 |
| **对工作的看法** | 找份稳定高薪的工作 | 建立能自动运转的系统 |
| **对风险的态度** | 避险，安全第一 | 管理风险，而不是回避风险 |
| **核心问题** | 「我买得起吗？」 | 「我怎样才能买得起？」 |
| **关于房子** | 房子是最大的投资 | 自住房是负债，不是投资 |
| **关于税收** | 老老实实交税 | 用公司结构合法节税 |
| **失败观** | 失败要避免 | 失败是学习的成本 |
| **最终结果** | 高薪但终生为钱奔波 | 让钱为他工作 |

**清崎不是说穷爸爸坏、富爸爸好**——两个爸爸都是好人。差别只在：**穷爸爸内化的是工业时代的雇员思维，富爸爸内化的是信息时代的所有者思维。** 学校教的全是前者，因为学校本来就是为培养雇员而设计的。

> 工程师视角：这一条对技术人极有杀伤力。我们受的训练是「成为更好的雇员」——刷题、跳槽、谈薪、晋升。清崎提醒：**这条路的终点是「高薪的雇员」，不是「自由的人」。** 雇员思维的天花板，是把时间换钱的效率优化到极致，而**时间的总量是有限的——所以这条路的天花板也是有限的。**

## 六堂课：清崎的核心教学

清崎把全书的精华浓缩成六堂课：

### 第 1 课：富人不为钱工作

穷人用时间换钱（打工）；中产用钱换负债（消费升级）；富人**用钱建立资产**，让资产去赚钱。

「为钱工作」的悲剧在于：**收入再高，时间也是有限的**——所以这条路有天花板。而「让钱工作」没有天花板，因为资产可以累积、可以复利、可以 24 小时不眠不休。

### 第 2 课：为什么要教授财务素养

清崎说：**学校教你赚钱，但从不教你管钱。** 这是工业时代教育系统的设计缺陷——它要的是听话的雇员，不是会理财的自由人。

他的结论简单粗暴：**你赚多少钱不重要，你能留住多少钱才重要。** 多数高薪人士最终也没什么财富，因为他们把赚到的全部换成了负债（大房子、好车、奢侈品）。

### 第 3 课：关注自己的事业

清崎区分了两个概念：

- **职业（Profession）**：你为之工作的东西，通常是别人的事业；
- **事业（Business）**：真正属于你、为你工作的资产。

打工族的职业是工程师，但他真正的事业可能是**他下班后建立的投资组合、写的产品、开的副业**。清崎要求每个人问自己一个问题：**「如果今天停止工作，我的资产还能养我多久？」** 这个问题的答案，就是你真实的财务状况。

### 第 4 课：税收的历史和公司的力量

清崎讲了一段税收简史：税收最初是向富人收的，但最终演变成**主要向中产阶级收**——因为富人用公司结构合法节税，穷人收入太低交得少，中产成了最大的税源。

核心差异：

- **雇员**：赚钱 → 交税 → 花剩下的；
- **公司**：赚钱 → 花掉该花的 → 只对剩下的交税。

清崎不是教逃税，是教**理解规则**：不同收入结构承担的税负完全不同。**你不需要成为会计专家，但你需要知道游戏规则是按结构设计的，不是按勤奋设计的。**

### 第 5 课：富人是发明的

清崎观察到：**穷人看到机会说「我买不起」，富人看到机会说「我怎样才能买得起？」**

前者关闭思维，后者启动思维。「我买不起」是个句号；「我怎样才能买得起」是个问号——问号会逼你去找杠杆、找资源、找创意。**财富不是发现的结果，是发明的过程。**

### 第 6 课：工作不是为了钱

清崎回忆自己年轻时免费给富爸爸打工，甚至倒贴钱学习。他的核心观点：**真正有价值的学习，往往不发生在「为了钱」的工作里。**

销售、谈判、领导力、财务常识——这些是致富的核心技能，但它们**不在任何岗位说明书里**，要靠主动接近、主动偷师。清崎建议：**别只看工资选工作，要看能学到什么技能。** 一份工资高但学不到东西的工作，是隐形的陷阱。

> 工程师视角：这一条对技术选型同样成立。选技术栈不该只看「市场价」，要看「能学到什么底层能力」。一份用时髦框架堆 CRUD 的高薪工作，远不如一份让你从 0 到 1 建系统、碰真实流量、解真实瓶颈的普通工作。

## 资产的四种类别

清崎把真正的资产分成四类，这是全书最具操作性的部分：

| 资产类别 | 例子 | 特点 |
|---------|------|------|
| **不需要自己在场的业务** | 公司、自动化副业 | 别人为你工作，系统自动运转 |
| **股票 / 基金** | 指数基金、分红股 | 流动性高，门槛低，被动 |
| **不动产** | 出租房（正现金流） | 能用杠杆，但管理成本高 |
| **知识产权** | 专利、版权、版税、软件 | 一次创作，长期收益 |

**清崎的建议：不要全压一类，要分散。** 每一类资产都有不同的风险收益特征和管理成本。早期可以先用门槛最低的（股票 / 基金）起步，逐步扩展到其他类别。

> 工程师视角：这就是「架构的多样性」。单一资产 = 单点故障；多类资产 = 异构冗余。**尤其第四类「知识产权」——代码、博客、开源项目、工具——对技术人是天然的资产赛道。** 一次创作，长期分红，零边际成本，这是技术人最被低估的财富机会。

## 老鼠赛跑（Rat Race）——全书最锋利的比喻

清崎用一个比喻贯穿全书：**老鼠赛跑**。

老鼠在轮子里拼命跑，跑得越快，轮子转得越快，但它永远在原地——这就是多数人的财务状态：

1. 加薪 → 消费升级（换车、换房、换学区）；
2. 开支上升 → 必须更努力工作保住收入；
3. 收入涨了但储蓄率没变 → 离自由反而更远；
4. 为了维持升级后的生活，更不敢离开工作 → 进入死循环。

**老鼠赛跑的核心特征：收入增加不仅没让人更自由，反而让人更不自由。** 因为生活方式的通胀速度，追平甚至超过了收入增速。

清崎的解药只有一条：**把涨薪的增量，优先变成资产，而不是变成消费。** 每涨一次薪，先把增量的一定比例投入资产账户，剩下的才升级生活。**让资产增长率永远跑赢消费增长率，是跳出老鼠赛跑的唯一公式。**

> 工程师视角：这就是「先扩容，再上负载」。服务器资源涨了，先把冗余和灾备做厚，而不是立刻让新业务把资源吃满。**技术债和财务债，是同一个数学问题。**

## 工程师视角（总览）

把清崎的观念翻译成工程师语言：

1. **资产 / 负债 = 正现金流 / 负现金流服务**：名义上线不算上线，**每月持续产出正向价值，才算真在线。**
2. **雇员思维 vs 所有者思维**：刷题跳槽谈薪是优化「时间换钱的效率」，天花板有限；建立资产是优化「钱换钱的效率」，天花板无限。
3. **老鼠赛跑 = 先上负载再扩容**：涨薪立刻消费升级 = 资源涨了立刻让业务吃满，冗余没增反减。
4. **关注自己的事业 = 建立下班后的资产**：职业是你为之工作的东西（别人的事业），事业是为你工作的资产（投资组合、副业、知识产权）。
5. **知识产权是技术人的天然赛道**：代码、博客、开源、工具——一次创作，长期分红，零边际成本。**这是技术人最被低估的财富机会。**

## 我的思考

### 1. 「自住房不是资产」颠覆了我过去十年的认知

读这本书最大的冲击，是清崎对自住房的判定——**它不是资产，是负债**。我过去一直把「买房」当成人生最大的资产配置，甚至为「上车」焦虑多年。清崎让我看清：如果这套房每月从你口袋抽走按揭、物业、维修，而你不靠它产生任何现金流，那它就是一份**几十年的负债合同**，不是资产。这不是说不该买房，而是说**别把负债误当成资产**——误判的代价，是错估了自己的财务自由进度。

### 2. 「关注自己的事业」治好了我的工作焦虑

我以前一直把「工程师」当成自己的事业，于是所有焦虑都来自工作——KPI、晋升、裁员传闻。清崎那句「你的职业是为别人工作的」像一记重锤：**我为之焦虑的，其实不是我的事业。** 真正属于我的事业，是我下班后建立的东西——投资组合、博客、开源、副业。从那天起，我把每天的时间重新分配：白天为职业工作，晚上为事业工作。**焦虑大幅下降，因为我开始为自己的资产工作，而不是为别人的 KPI 工作。**

### 3. 知识产权是技术人最被低估的赛道

清崎列的第四类资产——知识产权——对技术人是天然机会，但我过去完全没认真对待。写过的代码、发过的博客、做过的工具，都是**一次创作、长期分红、零边际成本**的资产，可我却把它们当一次性产出。读完这本书我开始重新审视：哪些我做过的事情，能变成持续产生现金流的资产？这是下一步最值得投入的方向——**不是再学一个新框架，而是把已有的能力封装成能复利的资产。**

## 行动清单

- [ ] 用清崎的现金流标准，重新清点自己的「资产 / 负债」清单——把每月净流出的项目标红
- [ ] 问自己一个问题并写下答案：「如果今天停止工作，我的资产能养我几个月？」——作为财务自由进度基准
- [ ] 区分「职业」和「事业」：写下白天为职业做的事，和晚上为事业做的事，确保后者不为零
- [ ] 每次涨薪，先把增量的固定比例（如 50%）投入资产账户，剩下的才升级生活
- [ ] 审视自己手上的「知识产权」：代码、文章、工具——哪些能封装成持续产生价值的资产
- [ ] 学习一项「不在岗位说明书里」的技能：销售、谈判、财务常识——每月至少投入 4 小时
- [ ] 每月一次「老鼠赛跑体检」：对比收入涨幅与消费涨幅，确保资产增长率跑赢消费增长率

## 一句话总结

> 穷人为钱工作，富人让钱为自己工作——中间隔的不是一个高薪，是一个叫「资产」的观念。
