# 《富有的习惯》读书笔记

> 富有不是某次暴击的运气，是日复一日复利的习惯。

---

## 关于本书

| | |
|---|---|
| **书名** | 《富有的习惯》（*Rich Habits*） |
| **作者** | 托马斯·科利（Thomas C. Corley） |
| **首版** | 2010 年 |
| **类型** | 个人理财 / 习惯研究 |

科利花了五年时间，调查 233 位富人（年净收入 ≥ 16 万美元、流动资产 ≥ 320 万）和 128 位穷人，对比他们的日常习惯。结论很朴素也很扎心：**贫富差距，本质上是习惯差距。** 富人不是某天变富的，是他们几十年积累的「富有习惯」把它们推到了那里。

这本书和《原子习惯》是同一路数，但更聚焦于**财富**这个具体领域，且基于实证调研而非纯理论。

## 核心框架：习惯是结果的前置变量

科利打了个比方：**习惯像飞往某地的航班。** 飞行途中 95% 时间是偏航的，飞行员不断微调，最终抵达目的地。

坏习惯像飞错方向，每天偏一点，几年后落在「穷」的地方；好习惯像飞对方向，每天修正一点，复利之下落在「富」的地方。

```
习惯（每日微小行为）
    │
    ├─ 好习惯 × 时间 = 富有 / 健康 / 成长
    │
    └─ 坏习惯 × 时间 = 贫穷 / 衰退 / 下滑
```

**核心公式**：富有 = 正确习惯 × 时间。时间是杠杆，习惯是支点。

## 富人的十大「富有习惯」

科利提炼出的高频共性，我挑最具冲击力的几条展开。

### 1. 坚持日常学习与自我提升

富人平均**每月读 2+ 本书**，内容集中在：行业知识、个人成长、理财、历史、传记。穷人几乎没有阅读习惯。

> 工程师视角：技术迭代快，持续学习本就是职业刚需。但富人读的不只是「专业书」，还读拓宽认知边界的书。**学习不是任务，是底层操作系统升级。**

### 2. 每天运动 30 分钟以上

76% 的富人每周有氧运动 4 天以上。运动提升精力、改善情绪、增强脑力——它是财富创造的「硬件基础」。

> 不要把运动当「额外负担」，它是「产能投资」。一个精力充沛的人和一个常年疲惫的人，十年后的差距是指数级的。

### 3. 设定目标并拆解

富人不是「希望变富」，而是写下具体的年度、月度、每日目标。**目标把模糊的「想要」变成可执行的「做到」。**

科利强调目标要可拆：年度目标 → 月度 → 周度 → 今天该做什么。没有落到「今天做什么」的目标，都是白日梦。

### 4. 每天做 80% 自己掌控的事

富人专注于**自己能控制**的领域（技能、健康、人脉、储蓄），而非被动等待运气（彩票、股市小道消息、贵人）。**掌控感 = 复利的基础。**

### 5. 建立并维护人际关系

富人主动维护人脉：记住别人生日、主动祝贺、定期联系、先给予再索取。**关系不是用的，是养的。**

### 6. 生活节俭，量入为出

这条最反直觉——多数富人**不显富**。他们住普通社区、开普通车、不追名牌。科利说：「富人买的不是身份，是资产。」穷人恰好相反：用消费营造富的假象，把钱花在贬值物上。

### 7. 不拖延——「现在就做」

富人有个口头禅：「Do it now」。拖延是穷人最普遍的习惯，它让计划永远停留在纸上。

### 8. 每天保持积极心态

富人刻意培养感恩、乐观、自信。他们相信「我命由我」，而非「命运由天」。心态影响行为选择，行为选择决定结果。

### 9. 储蓄优先，消费在后

富人先存钱再花钱（**Pay yourself first**），通常把收入 10-20% 强制储蓄/投资。穷人是先花钱再存剩下的——而「剩下的」几乎永远是零。

### 10. 控制情绪与言辞

富人少抱怨、少八卦、少说绝对话。他们明白：**情绪化和口无遮拦会破坏关系和机会。**

## 「穷习惯」对照

科利同样列出了穷人的高频坏习惯，核心几条：

| 穷习惯 | 后果 |
|--------|------|
| 过度刷屏（电视/短视频/社交媒体） | 时间被消费，注意力被收割 |
| 冲动消费、超前消费 | 永远在还债，无法积累资本 |
| 不运动、饮食不健康 | 健康债务，未来医疗支出暴涨 |
| 不读书、停止学习 | 认知固化，被时代淘汰 |
| 怨天尤人、受害者心态 | 失去掌控感，行动瘫痪 |
| 只盯着眼前、不做长期规划 | 永远在救火，无法复利 |

## 我的思考

### 1. 这本书戳破了一个幻觉

我曾以为「富有」=「赚到一大笔钱」——某次创业成功、某支股票暴涨、某个项目分红。读完这本书才意识到：**那是少数幸存者，多数富人是缓慢积累出来的。** 他们每天做的小事，我每天也在做（或没做）。差距不是某次大跃进，是每天 1% 的方向。

### 2. 贫穷是「系统性」的，不是道德问题

科利的研究让我看到：穷人不是「不够努力」，而是陷在一套**坏习惯系统**里。每天刷 4 小时短视频、靠外卖度日、月光不储蓄——这些行为在当下感觉「还好」，但它们是负复利，时间越久越被锁死。

改变不是靠意志力硬扛，而是**逐个替换习惯**。把一个坏习惯换成好习惯，持续 30 天，再换下一个。**系统比意志力可靠。**

### 3. 对工程师的特别启发

科利没专门讲程序员，但几条对工程师尤其刺眼：

- **健康是第一资产**：久坐、熬夜、外卖——这是工程师最普遍的「穷习惯」，健康债务比信用卡债务更危险，因为它沉默积累、一次性爆发。
- **储蓄优先 = 技术债务优先**：就像代码里不能只堆功能不还技术债，财务上也不能只消费不储蓄。
- **持续学习是真正的「富有习惯」**：在这个行业，停止学习 = 自动贬值。

## 一句话总结

> 你不是因为富有才有好习惯，而是因为有了好习惯才会富有。习惯是因，财富是果。

## 延伸

- **《原子习惯》**（James Clear）：习惯养成的科学方法论，和本书互补——一个讲「为什么」，一个讲「怎么做」。
- **《富爸爸穷爸爸》**（清崎）：资产 vs 负债的核心观念，与本书「买资产不买身份」高度一致。
- **《财务自由之路》**（博多·舍费尔）：欧洲视角的理财习惯书，强调储蓄率和复利。
- **《掌控习惯》实操**：把科利的「富有习惯」用克利尔的四定律（提示/渴求/反应/奖励）落地。
