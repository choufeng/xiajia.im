# 《毛泽东选集》精读系列

> 《毛选》不是政治符号，而是一座**方法论的富矿**。教员一生打仗、搞建设、带队伍，留下的不是口号，是一套套「怎么认识世界、怎么改变世界」的硬方法。
>
> 这个系列不按卷册顺序通读，而是**按方法主题切片**——把散在《实践论》《矛盾论》《论持久战》《反对本本主义》等名篇里的同类思想聚到一起，看清这套方法论的骨架。

---

## 为什么按方法主题读

《毛选》四卷（一至四卷，收录 1925–1949 年著作）文章多、跨度大，按时间顺序读容易「只见篇章不见体系」。但教员的方法论高度一致，贯穿几条主线：

1. **怎么认识**：实践论——知识从实践中来，又回到实践检验。
2. **怎么分析**：矛盾论——抓主要矛盾，具体问题具体分析。
3. **怎么做事**：调查研究、集中优势兵力、分阶段。
4. **怎么对人**：群众路线、调查研究、团结—批评—团结。

把它们**按方法重新聚合**，能看清这套「认识—分析—行动」的完整闭环，也能照见今天做事、带团队、做工程时仍然好用的大智慧。

## 精读体例

每篇遵循统一结构：

- **写作背景**：这篇文献「何时、为何、针对什么」而写——理解方法离不开语境。
- **原文摘录**：摘录该主题 4–6 条核心论断，标注出处篇名。
- **逐段精解**：每条 → 还原论点 → 精解其方法内核 → 当代映射。
- **思想坐标**：该方法在教员思想体系里的位置，与前后文献如何呼应。
- **当代映射**：这套方法今天怎么用——尤其对工程师、管理者、做产品的人。
- **个人体悟**：第一人称的实在感受，不吹不黑。

## 一点说明

读《毛选》的方法论，是把它当**思想工具**来看待——哪些经得起时间检验、今天依然锋利，哪些受限于历史语境。本系列取前者：那些关于「如何认识、如何分析、如何行动」的普遍智慧。
