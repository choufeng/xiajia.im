# 《道德经》精读系列

> 五千言，八十一章，把「天地怎么运行、人该怎么活、事该怎么做」讲到了底。
>
> 这个系列不按原书八十一章顺序逐章通读，而是**按主题切片**——把散落在各章里、讲同一件事的章句聚到一起，看清老子思想的内在骨架。

---

## 为什么按主题读

《道德经》八十一章，似格言集，章与章之间无明显脉络。按原书顺序读容易「只见金句不见体系」。老子反复讲「道」「无为」「柔弱」「反」「朴」，这些概念散在数十章里。

把它们**按主题重新聚合**，三个好处：

1. **看清骨架**：一个概念在全书如何展开、如何自洽。
2. **便于对照**：同主题章句放一起，老子在不同章里反复申说的同一层意思，反而显出深意。
3. **落到当代**：主题切片天然适合「这个道理今天怎么用」。

## 精读体例

每篇遵循统一结构：

- **缘起**：为何选这个主题，它在全书的位置。
- **原文精选**：摘录该主题 4–6 条经典章句，标注标准编号（章序，如「第一章」）。
- **逐句精解**：每条 → 白话译 → 精解 → 当代映射。这是每篇核心。
- **一以贯之**：提炼该主题在老子思想里的内在逻辑。
- **当代映射**：落到今天，尤其给工程师 / 知识工作者的启发。
- **个人体悟**：第一人称的实在感受。

## 版本与体例说明

- 原文以王弼通行本为准（参陈鼓应《老子今注今译》、任继愈《老子绎读》）。
- 章句编号采用八十一章标准顺序（如「第一章」「第八章」），便于检索。
- 译文力求准确直白，不取过度演义。
