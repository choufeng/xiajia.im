# 《非暴力沟通实践篇》读书笔记

> 主书教公式，实践篇教落地。知道「观察—感受—需要—请求」是一回事，在愤怒、冲突、亲子、自我攻击里还能用出来，是另一回事。

---

## 关于本书

| | |
|---|---|
| **书名** | 《非暴力沟通实践篇》（*Nonviolent Communication: A Companion Workbook* / *Speak Peace in a World of Conflict*） |
| **作者** | 马歇尔·卢森堡（Marshall B. Rosenberg） |
| **出版** | 江苏人民出版社，2014（中文版） |
| **类型** | 沟通 / 实操 / 冲突调解 |

这是《非暴力沟通》的姊妹篇。主书建立理论框架（四要素、异化语言、倾听），实践篇把这套语言**塞进真实场景**——愤怒的转化、冲突调解、亲子教育、亲密关系、自我对话、表达感激、强制力的边界。

卢森堡在序言里反复强调一句话：**NVC 不是沟通技巧，是一种生活方式**。目标不是「把话说漂亮」，而是**丰盈生命**（life-enriching）——让自己和他人的需要都被看见、被照顾。

> 如果你还没读过主书，强烈建议先读 [《非暴力沟通》](./nonviolent-communication)。本文假设你已熟悉四要素框架，不再展开。

## 定位：主书讲「道」，实践篇讲「用」

| 维度 | 主书 | 实践篇 |
|------|------|--------|
| 重点 | 建立意识：观察 vs 评判、需要 vs 策略 | 场景落地：愤怒、冲突、亲子、感激、自我 |
| 叙事 | 概念 + 经典案例 | 大量真实工作坊对话 |
| 难点 | 换思维 | 在情绪上头时仍守住 NVC |
| 读法 | 通读 | 当工具书，按场景查 |

一句话：主书告诉你**为什么**，实践篇告诉你**什么时候用、怎么用出来**。

## 公式复习

```
当我看到/听到 ____（观察），
我感到 ____（感受），
因为我需要 ____（需要）。
你是否愿意 ____（请求）？
```

四步缺一不可，顺序不能乱。实践篇的全部场景，都是对这四步的反复演练。下面进入实践篇的真正增量。

## 核心场景一：转化愤怒

这是实践篇最重要的章节，也是 NVC 最反直觉的地方。

### 愤怒不是别人「惹」出来的

主流叙事：他骂我 → 我生气。因果在外。

卢森堡的翻转：**别人做的事只是刺激，愤怒的真正原因是你脑子里的那句评判**。

> 同事在会上打断你。
> - 评判版：「这人真没教养，故意让我难堪！」→ 暴怒
> - NVC 版：「他打断了我三次（观察）。我感到沮丧（感受），因为我需要被完整听见（需要）。」

**同一个事件，两种情绪反应，差别全在你怎么归因。** 评判制造愤怒，需要带来建设性能量。

### 愤怒处理四步

1. **停下来，深呼吸**——别在风暴中心开口
2. **找出脑子里那句评判**——「他怎么可以」「他太……」
3. **把评判翻译成需要**——这个评判背后，我什么需要没被满足？
4. **开口时只说需要和请求**，不说评判

### 「应该」是给自己的暴力

愤怒思维里最隐蔽的一类是「**他应该……**」「**人不应该这样**」。卢森堡把它们叫**敌对型思维**（enemy images）。每养一个敌对形象，就给自己的愤怒添一把柴。

> 把「他应该准时」换成「我希望他准时，因为准时对我来说意味着尊重」。
> 前者是评判，后者是清楚的需要。前者制造冲突，后者打开对话。

## 核心场景二：冲突调解

NVC 的冲突调解，和「仲裁」「说理」完全不同。

### 第一步：先连接人，再讨论事

传统调解：摆事实、讲道理、判对错。
NVC 调解：**让双方先把彼此当成「人」，而不是「问题」**。

> 卢森堡在巴以冲突、种族骚乱现场做的事，永远是先让双方说出：「**你希望被听见的是什么？什么对你最重要？**」

### 第二步：翻译评判性语言

冲突中的语言几乎全是评判——「他自私」「她不负责任」。调解者的工作是**当翻译**：

| 当事人原话 | 翻译成需要 |
|-----------|-----------|
| 「他根本不在乎这个家」 | 你需要**共同参与**和**被重视**？ |
| 「她总是控制我」 | 你需要**自主权**和**被信任**？ |
| 「他太自私了」 | 你需要**相互体谅**和**公平**？ |

一旦评判被翻译成需要，对立瞬间松动——因为**所有人的需要都是正当的**，冲突只在「策略」层面，不在「需要」层面。

### 第三步：用当下可执行的请求替代「谁对谁错」

调解失败的常见原因：双方争论过去的对错，谁也不肯让。
NVC 的出路：**不谈过去，只谈未来——你希望对方接下来具体做什么？**

> 「我希望你更关心我」——太模糊，不可执行。
> 「每天睡前花十分钟聊聊今天发生的事」——具体，可执行，可验收。

### 请求 vs 命令（在冲突中尤为关键）

冲突里最危险的伪装：把命令说成请求。
判别标准不变——**对方拒绝后你会惩罚、冷战吗？** 会，就是命令。

真正的 NVC 调解，允许双方都说「不」，然后在「不」的基础上继续找方案。**没有「不」的权利，就没有真正的合作。**

## 核心场景三：自我宽恕

这是常被忽略、却极其重要的一章。NVC 不只对别人，更要对自己。

### 「我应该」是自我暴力

我们对自己说的话，往往比对任何人都狠：

> 「我真蠢，又搞砸了。」
> 「我必须改掉这个毛病。」
> 「我应该更努力。」

每一句「我应该」，都是一次自我评判。**对自己用 NVC，意味着把对自己的暴力也停下来。**

### 把「我应该」换成「我选择」

卢森堡的方法：**把每个「我应该」改成「我选择做 X，是为了满足 Y 需要」**。

> 「我应该减肥」
> → 「我选择控制饮食，是因为我需要健康和精力。」

> 「我不得不加班」
> → 「我选择加班，是因为我需要项目按时上线，也需要对团队负责。」

这不是自我欺骗，是**拿回选择权**。一旦意识到这是「选择」而非「被迫」，怨气会消大半，行动力反而上升。

### 对「过去的自己」也用 NVC

> 不是「我当年真傻」，而是「当时的我，是在用我那时的认知和资源，满足我当时最在意的需要。现在的我有了新的理解，可以做不同的选择。**

**自我攻击不会让人变好，只会让人更僵。** 对自己温柔，不是放纵，是给改变留出空间。

## 核心场景四：表达感激

主书讲过评价式赞美的陷阱，实践篇把感激讲得更细。

### 感激三步

1. **对方做了什么**（具体行为，不是「你真好」）
2. **满足了你的哪个需要**
3. **你此刻什么感受**

> 评价式：「你真棒，帮了大忙！」
> NVC：「你昨天主动把测试补全了（行为），这让我对上线有了安全感（满足的需要），我感到很踏实、也很感激（感受）。」

### 为什么 NVC 感激更有力

- 评价式赞美 = 居高临下打分，让人依赖外部认可
- NVC 感激 = 平等地分享「你的行为如何丰盈了我的生命」，让人知道**做对了什么、为什么有价值**

前者培养讨好型人格，后者培养内在动机。

### 接受感激也要练习

很多人被夸时本能否认：「哪里哪里，没什么。」这其实是在**阻断连接**。
NVC 建议：**全然接受**——「听到你这么说，我很高兴。」让对方的好意真正抵达你。

## 核心场景五：倾听的深度

主书讲过「翻译攻击」，实践篇补充几个实操要点。

### 倾听 ≠ 给建议

大多数「倾听」其实是**等着轮到自己说话**，然后立刻给建议、讲道理、分享自己的经历。
真正的倾听是：**暂时放下自己，进入对方的世界**。

### 三种倾听动作

1. **沉默陪伴**——有时不说话就是最大的连接
2. **复述确认**——「你刚才说……我理解得对吗？」
3. **反射感受和需要**——「听起来你很沮丧，是因为你需要被认可？」

### 何时停止倾听

卢森堡给出一个信号：**当你感到自己开始烦躁、想反驳，说明你的需要没被照顾，这时候要先照顾自己**。NVC 不要求牺牲自己成全别人——**先戴好自己的氧气面罩**。

## 核心场景六：强制力的边界

这是 NVC 最容易招误解的部分——「那孩子要跑到马路上也不管？对方打人也不阻止？」

卢森堡的区分极清晰：

| | 保护性强制力 | 惩罚性强制力 |
|---|---|---|
| 动机 | 阻止伤害，保护需要 | 让对方「付出代价」 |
| 时机 | 当下、必要 | 事后、报复 |
| 心态 | 「我必须阻止这件事」 | 「你活该」 |
| 效果 | 危险解除 | 制造恐惧和敌意，埋下新冲突 |

**判断标准：你的强制力是为了保护，还是为了对方「受教训」？**
- 拉住要冲向马路的孩子 = 保护
- 打孩子让他「长记性」= 惩罚

惩罚性强制力的问题不在于「严」，而在于**它无法满足任何人的需要**——对方学到的只是「下次别被抓到」，不是「为什么不该这样做」。

## 关键认知翻转（实践篇的核心增量）

把实践篇最有冲击力的认知，浓缩成六条：

1. **别人的话让你难受，是「评判」让你难受，不是话本身** → 拆评判，找需要
2. **「应该」是暴力**——对别人是命令，对自己是自我攻击 → 换成「我选择」
3. **诊断重于连接，是暴力的起点** → 先连结人，再讨论事
4. **请求 ≠ 命令**：被拒后能继续对话 = 请求；被拒就惩罚/冷战 = 命令
5. **自我攻击不会让人变好** → 对过去的自己用 NVC
6. **惩罚无法满足任何需要** → 只用保护性强制力，不用惩罚性强制力

## 落地三问

任何冲突、情绪上头、要开口前，立刻问自己：

1. 我此刻的**观察**是什么？（去掉「总是」「从不」）
2. 我此刻的**感受**和**需要**是什么？
3. 我希望对方**具体做什么**？（不是「理解我」「对我好点」这种不可执行请求）

> **工程师视角**：代码评审里最值钱的 NVC 套用——
> 「这个函数有 5 个职责、200 行（观察），我担心后续难以维护（感受），因为我需要可维护性（需要），能否拆成三个函数（请求）？」
> 把被告席上的人，拉回工位。

## 我的思考

### 1. 实践篇真正难的不是技巧，是情绪上头时还守得住

主书读完，我以为我懂了。实践篇读完，才发现**懂和做到之间隔着一整个情绪管理**。NVC 在冷静时谁都会用，难的是被人当众羞辱、孩子第三次打翻牛奶、需求方反复改需求时——那 0.5 秒里还能不能想起「找评判，找需要」。实践篇的价值，就是把这些场景反复演练到形成肌肉记忆。

### 2. 「我应该」那一章击中了我

我对自己说的话，刻薄到对任何朋友说都会绝交的程度。「你又拖延了」「你这点事都做不好」。读实践篇才发现，**我一直在用暴力对自己**，还美其名曰「严格要求」。把「我应该」改成「我选择」之后，行动力反而上升——因为负担从「被迫」变成了「主动」。这是全书对我改变最大的一章。

### 3. 冲突调解的「翻译评判」可迁移到任何对话

把对方攻击性的话翻译成背后的需要——这个动作不只用于冲突，**日常沟通里也极其好用**。同事抱怨「这需求真扯淡」，翻译成「他可能需要需求更清晰、更稳定」。一旦养成翻译习惯，对立感大幅下降，对话空间立刻打开。

### 4. NVC 不是让你变软，是让你变清晰

实践篇彻底打消了我对 NVC「和稀泥」的担忧。它要求你**比任何人都更清楚自己的需要**，只是不用攻击的方式表达。在守住自己需要的同时不伤害关系——这才是真正的力量。不是压服别人，是清晰和诚实赢得合作。

## 行动清单

- [ ] 情绪上头时先停 3 秒，找脑子里的那句「评判」
- [ ] 把每个「他应该 / 我应该」改写成「我选择 X，是为了 Y」
- [ ] 冲突里不当裁判，当翻译——把评判翻成需要
- [ ] 开口前自检：我说的是观察还是评论？是请求还是命令？
- [ ] 赞美别人用「行为 + 满足的需要 + 感受」三步法
- [ ] 接受赞美时不否认，说「听到你这么说我很高兴」
- [ ] 对自己出错时，用 NVC 而非自我攻击
- [ ] 任何强制力前自问：这是保护，还是惩罚？

## 一句话总结

> 懂公式不难，难的是愤怒、冲突、自我攻击里仍守得住「观察、感受、需要、请求」。NVC 不是话术，是一种让自己和他人都被看见的生活方式。

## 延伸

- **[《非暴力沟通》](./nonviolent-communication)**（卢森堡）：主书，建立四要素框架与意识，本文的前置阅读。
- **[《高效能人士的七个习惯》—— 知彼解己](./seven-habits)**：习惯五「先理解别人，再求被理解」与 NVC 倾听同源。
- **《非暴力沟通亲子篇》**（卢森堡）：NVC 在家庭教育中的具体应用，和本书「强制力」「倾听」章节互补。
- **《非暴力沟通实践手册：爱的语言练习本》**（Lucy Leu）：13 周刻意练习册，配合主书与实践篇使用，把 NVC 练成肌肉记忆。注意此书为练习册，非本「实践篇」。
- **《关键对话》**（Kerry Patterson 等）：高压高风险场景的沟通术，NVC 讲意识，本书讲危机应对。
